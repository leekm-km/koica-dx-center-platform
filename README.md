# KOICA DX Center Platform

**Philippines Digital Transformation Platform**

필리핀 KOICA DX Center에서 개발한 디지털 전환 플랫폼입니다.

## 📁 프로젝트 구조

```
koica-dx-center-platform/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css      # 스타일시트
├── js/
│   └── main.js        # JavaScript 파일
├── assets/
│   └── images/        # 이미지 리소스
└── README.md          # 프로젝트 문서
```

## 🚀 시작하기

### 로컬에서 실행

웹 페이지를 실행하려면 `index.html` 파일을 브라우저에서 열거나, 로컬 서버를 실행하세요:

#### Python 사용
```bash
python3 -m http.server 8000
```

#### Node.js 사용
```bash
npx http-server -p 8000
```

그 후 브라우저에서 `http://localhost:8000`을 엽니다.

## 🎨 디자인 시스템

프로젝트는 KOICA 브랜드 컬러를 사용한 디자인 시스템을 포함합니다:
- **Primary Color**: #0066cc (KOICA Blue)
- **Secondary Color**: #00a8e8 (Sky Blue)
- **다크 모드 기본 적용**
- **그라디언트 효과 및 애니메이션**
- **반응형 레이아웃**

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: CSS 변수, 그라디언트, 애니메이션
- **Vanilla JavaScript**: 프레임워크 없는 순수 JavaScript
